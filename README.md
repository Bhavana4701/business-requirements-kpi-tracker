# Business Requirements & KPI Dashboard Tracker

## Overview
A business analysis project simulating requirements gathering, gap analysis, 
and KPI tracking across 3 operational initiatives. Built to demonstrate 
end-to-end BA documentation and data reporting workflows.

## Business Problem
Organizations struggle to track multiple active initiatives while maintaining 
clear stakeholder alignment and measurable KPIs. This project simulates a 
structured BA approach to solve that.

## What's Included
- Business Requirements Document (BRD) template
- Gap analysis report identifying 5 process inefficiencies
- SQL queries for KPI extraction and trend analysis
- Power BI dashboard mockup (PDF export)
- Sprint tracking log using Agile/Scrum format

## Tools Used
SQL | Excel | Power BI | JIRA (simulated) | Confluence (simulated)

## Key Outcomes
- Documented 10+ business processes with BRDs and FRDs
- Identified inefficiencies projected to improve efficiency by 30%+
- Built KPI dashboards covering operational metrics across 4 teams
